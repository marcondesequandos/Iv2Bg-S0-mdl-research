Material Design Lite

MDL é framework que ajuda na elaboração do design dos sites, tanto no front end quanto para painéis administrativos. Oferece opções com um design leve para uma série de componentes fundamentais para o design e usabilidade de uma página web como: ícones, botões, cards, menus, cabeçalhos, tabelas etc.
